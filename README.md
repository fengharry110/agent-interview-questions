# Agent 面试题库

这是一个面向 **Agent 工程师求职面试** 的文档型项目。

项目目标不是做一个工具，而是沉淀一套系统化、可复习、可追问、可持续扩展的 Agent 面试知识库。每一个问题都是独立的 Markdown 文件，方便按主题学习，也方便在面试前快速回顾。

## 内容结构

```text
面试题库/
  目录.md
  01-智能体基础/
  02-工具调用/
  03-检索增强生成/
  04-评测与调试/
  05-安全与权限/
  06-工程化与生产/
```

每个问题文件都包含：

- 面试问题
- 考察重点
- 核心结论
- 参考回答
- 深度展开
- 工程实践
- 常见误区
- 追问方向
- 学习建议

## 适合谁

- 想求职 Agent 工程师的新手
- 想系统理解 LLM Agent 的开发者
- 想准备 LLM 应用 / RAG / Tool Calling 面试的人
- 想把 Agent 知识整理成 GitHub 作品集的人

## 推荐阅读顺序

1. 先读 `01-智能体基础`
2. 再读 `02-工具调用`
3. 然后读 `03-检索增强生成`
4. 接着读 `04-评测与调试`
5. 最后读 `05-安全与权限` 和 `06-工程化与生产`
