# Agent 面试题库

这是一个面向 **Agent 工程师求职面试** 的中文文档型项目。

项目目标不是做一个工具，而是沉淀一套系统化、可复习、可追问、可持续扩展的 Agent 面试知识库。每一个问题都是独立的 Markdown 文件，方便按主题学习，也方便在面试前快速回顾。

## 内容结构

```text
面试题库/
  目录.md
  01-智能体基础/
  02-意图识别与任务理解/
  03-任务规划与执行控制/
  04-工具调用/
  05-上下文工程/
  06-检索增强生成/
  07-可观测性与调试/
  08-评测体系/
  09-安全与权限/
  10-多智能体系统/
  11-生产化工程/
  12-业务场景与系统设计题/
  13-简历与项目表达/
```

每个问题文件都包含：

- 面试问题
- 考察重点
- 核心结论
- 参考回答
- 深度展开
- 工程实践
- 常见误区
- 追问方向
- 学习建议

## 适合谁

- 想求职 Agent 工程师的新手
- 想系统理解 LLM Agent 的开发者
- 想准备 LLM 应用、RAG、工具调用、评测和生产化面试的人
- 想把 Agent 知识整理成长期作品集的人

## 推荐阅读顺序

1. 先读智能体基础、意图识别、任务规划
2. 再读工具调用、上下文工程、检索增强生成
3. 接着读可观测性、评测体系、安全与权限
4. 最后读多智能体、生产化工程、系统设计和项目表达
