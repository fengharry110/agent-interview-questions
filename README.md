# Agent Interview Questions

这是一个面向 **Agent 工程师求职面试** 的文档型项目。

项目目标不是做一个工具，而是沉淀一套系统化的 Agent 面试题库。每一个问题都是独立的 Markdown 文件，方便阅读、复习、扩展，也方便后续上传到 GitHub。

## 内容结构

```text
docs/
  01-agent-basics/
  02-tool-calling/
  03-rag/
  04-evaluation/
  05-safety/
  06-engineering/
```

每个问题文件都包含：

- 面试问题
- 考察重点
- 参考回答
- 追问方向
- 学习建议

## 适合谁

- 想求职 Agent 工程师的新手
- 想系统理解 LLM Agent 的开发者
- 想准备 LLM 应用 / RAG / Tool Calling 面试的人
- 想把 Agent 知识整理成 GitHub 作品集的人

## 推荐阅读顺序

1. 先读 `01-agent-basics`
2. 再读 `02-tool-calling`
3. 然后读 `03-rag`
4. 接着读 `04-evaluation`
5. 最后读 `05-safety` 和 `06-engineering`
