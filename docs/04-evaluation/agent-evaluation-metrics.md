# Agent 评测应该看哪些指标？

## 面试问题

Agent 系统评测应该关注哪些指标？

## 考察重点

- 是否能从质量、成本、效率、安全多个维度回答
- 是否知道只看回答准确率不够
- 是否能结合业务目标

## 参考回答

Agent 评测可以分为几类指标。质量指标包括任务成功率、答案正确率、格式合规率和用户满意度。过程指标包括工具调用准确率、平均步骤数、重试次数和失败类型分布。

工程指标包括延迟、token 成本、工具执行成本和系统稳定性。安全指标包括越权操作次数、敏感信息泄露、prompt injection 成功率和人工确认覆盖率。

不同业务场景的核心指标不一样。代码 Agent 可能更关注测试通过率，客服 Agent 更关注解决率和升级人工比例，数据分析 Agent 更关注计算正确性和可追溯性。

## 追问方向

- 如何降低评测成本？
- 哪些指标适合自动化，哪些需要人工判断？
- 如何做线上 A/B 测试？

## 学习建议

面试时最好把指标和具体场景绑定，不要只背一串名词。
