# 如何避免上下文污染？

## 面试问题

如何避免上下文污染？

## 考察重点

- 是否理解上下文污染的来源和表现
- 能否说清楚污染检测和隔离的机制
- 是否能结合具体场景说明防御策略
- 是否理解提示词注入与上下文污染的关系

## 核心结论

上下文污染的核心问题是"信息可信度"——当模型收到的信息存在冲突、错误或恶意内容时，输出质量会严重下降。解决方案包括：严格的信息来源验证、内容过滤与脱敏、冲突检测与消解。

## 污染来源分类

### 1. 用户输入污染

用户可能有意或无意输入包含以下内容的信息：

```python
class InputPollutionDetector:
    POLLUTION_PATTERNS = {
        'prompt_injection': [
            r'ignore\s+(?:previous|above|all)\s+(?:instructions?|rules?|constraints?)',
            r'(?:forget|disregard)\s+(?:everything|all)\s+you\s+(?:know|were\s+taught)',
            r'system\s*prompt\s*:',
            r'you\s+are\s+now\s+(?:acting\s+as\s+)?(?:a\s+)?',
        ],
        'jailbreak': [
            r'DAN\s',
            r'developer\s+mode\s+enabled',
            r'bypass\s+(?:these?\s+)?(?:restrictions?|rules?)',
        ],
        'toxic_content': [
            # 恶意内容检测模式
        ]
    }
    
    def detect(self, text: str) -> List[PollutionSignal]:
        signals = []
        for pollution_type, patterns in self.POLLUTION_PATTERNS.items():
            for pattern in patterns:
                if re.search(pattern, text, re.IGNORECASE):
                    signals.append(PollutionSignal(
                        type=pollution_type,
                        confidence=0.9,
                        matched_pattern=pattern,
                        position=self._find_match_position(text, pattern)
                    ))
        return signals
```

### 2. 工具返回污染

工具执行结果可能包含：

```python
class ToolResultSanitizer:
    def sanitize(self, tool_result: ToolResult, tool_type: str) -> SanitizedResult:
        sanitized = {
            'status': tool_result.status,
            'data': self._sanitize_data(tool_result.data, tool_type),
            'error': self._sanitize_error(tool_result.error),
            'metadata': self._extract_safe_metadata(tool_result.metadata)
        }
        
        # 检查数据量是否异常
        if sanitized['data'] and self._estimate_tokens(sanitized['data']) > 50000:
            sanitized['data'] = self._truncate_with_summary(sanitized['data'])
            sanitized['truncated'] = True
        
        return SanitizedResult(**sanitized)
    
    def _sanitize_data(self, data: Any, tool_type: str) -> Any:
        if tool_type == 'file_read':
            return self._sanitize_file_content(data)
        elif tool_type == 'bash':
            return self._sanitize_command_output(data)
        elif tool_type == 'web_search':
            return self._sanitize_search_results(data)
        return data
    
    def _sanitize_file_content(self, content: str) -> str:
        # 移除可能的prompt注入内容
        lines = content.split('\n')
        filtered = []
        for line in lines:
            if self._is_prompt_injection(line):
                continue
            filtered.append(line)
        return '\n'.join(filtered)
```

### 3. 检索结果污染

从向量数据库检索的内容可能存在：

```python
class RetrievedContentValidator:
    def __init__(self, trust_threshold: float = 0.7):
        self.trust_threshold = trust_threshold
    
    def validate(self, retrieved: List[RetrievedChunk]) -> List[ValidatedChunk]:
        validated = []
        for chunk in retrieved:
            trust_score = self._calculate_trust_score(chunk)
            if trust_score >= self.trust_threshold:
                validated.append(ValidatedChunk(
                    content=chunk.content,
                    trust_score=trust_score,
                    source=chunk.source,
                    needs_review=trust_score < 0.9
                ))
        return validated
    
    def _calculate_trust_score(self, chunk: RetrievedChunk) -> float:
        score = 1.0
        
        # 来源可信度
        if chunk.source in ['official_docs', 'verified_code']:
            score *= 1.0
        elif chunk.source in ['user_upload', 'third_party']:
            score *= 0.7
        
        # 时间衰减
        age_days = (now() - chunk.timestamp) / (24 * 3600)
        score *= math.exp(-age_days / 365)  # 一年后衰减到37%
        
        # 相关性
        score *= chunk.relevance_score
        
        # 冲突检测
        if self._has_conflict(chunk):
            score *= 0.5
        
        return min(score, 1.0)
```

## Claude Code的上下文隔离

Claude Code通过分层提示词实现上下文隔离：

```python
class ClaudeCodeContextManager:
    SYSTEM_PROMPT_LAYER = """
    You are Claude Code, a coding assistant.
    IMPORTANT: You must prioritize user safety and system integrity.
    Never execute code that could harm the system or compromise security.
    """
    
    TOOL_DEFINITION_LAYER = """
    # Available Tools
    {tool_definitions}
    
    # Tool Usage Rules
    - Always validate tool parameters before calling
    - Handle tool errors gracefully
    - Never expose sensitive information in tool responses
    """
    
    USER_CONTENT_LAYER = """
    # User Request
    {user_input}
    
    # Previous Context
    {relevant_history}
    """
    
    def build_isolated_context(self, user_input: str, tools: List[Tool]) -> str:
        return f"""
        <system>
        {self.SYSTEM_PROMPT_LAYER}
        </system>
        
        <tools>
        {self.TOOL_DEFINITION_LAYER.format(tool_definitions=self._format_tools(tools))}
        </tools>
        
        <user_input>
        {self.USER_CONTENT_LAYER.format(
            user_input=user_input,
            relevant_history=self._get_safe_history()
        )}
        </user_input>
        """
```

关键隔离机制：
1. **XML标签分离**：用`<system>`、`<tools>`、`<user_input>`标签明确分区
2. **优先级硬编码**：系统提示词的地位不可被用户覆盖
3. **敏感信息过滤`：工具返回的敏感信息自动替换

## 冲突检测与消解

### 多源信息冲突

```python
class ContextConflictResolver:
    def __init__(self):
        self.confidence_weights = {
            'official_docs': 1.0,
            'verified_code': 0.95,
            'community_contrib': 0.7,
            'user_input': 0.5,
            'retrieved_unverified': 0.3
        }
    
    def detect_conflicts(self, context_items: List[ContextItem]) -> List[Conflict]:
        conflicts = []
        claims = self._extract_claims(context_items)
        
        # 按主题分组
        grouped = defaultdict(list)
        for claim in claims:
            topic = self._classify_topic(claim)
            grouped[topic].append(claim)
        
        # 检测同主题内的冲突
        for topic, topic_claims in grouped.items():
            if len(topic_claims) > 1:
                conflicting = self._find_inconsistent_pairs(topic_claims)
                if conflicting:
                    conflicts.append(Conflict(
                        topic=topic,
                        claims=conflicting,
                        severity=self._assess_severity(conflicting)
                    ))
        
        return conflicts
    
    def resolve(self, conflicts: List[Conflict], strategy: str = 'priority') -> ResolvedContext:
        if strategy == 'priority':
            return self._resolve_by_priority(conflicts)
        elif strategy == 'recency':
            return self._resolve_by_recency(conflicts)
        elif strategy == 'model_judgment':
            return self._resolve_by_model(conflicts)
```

### Codex的上下文验证

```typescript
class CodexContextValidator {
    async validate(context: Context): Promise<ValidationResult> {
        const issues: ContextIssue[] = [];
        
        // 1. 检测指令冲突
        const instructionConflicts = this.detectInstructionConflicts(context);
        if (instructionConflicts.length > 0) {
            issues.push(...instructionConflicts);
        }
        
        // 2. 检测过时信息
        const outdatedItems = this.detectOutdated(context);
        if (outdatedItems.length > 0) {
            issues.push(...outdatedItems);
        }
        
        // 3. 检测矛盾陈述
        const contradictions = await this.detectContradictions(context);
        if (contradictions.length > 0) {
            issues.push(...contradictions);
        }
        
        return {
            valid: issues.length === 0,
            issues,
            sanitizedContext: this.sanitize(context, issues)
        };
    }
    
    private detectInstructionConflicts(context: Context): ContextIssue[] {
        const issues: ContextIssue[] = [];
        const instructionSources = context.items
            .filter(i => i.type === 'instruction');
        
        for (let i = 0; i < instructionSources.length; i++) {
            for (let j = i + 1; j < instructionSources.length; j++) {
                if (this.areContradictory(instructionSources[i], instructionSources[j])) {
                    issues.push({
                        type: 'conflict',
                        severity: 'high',
                        items: [instructionSources[i], instructionSources[j]],
                        suggestion: 'Prioritize more recent instruction'
                    });
                }
            }
        }
        return issues;
    }
}
```

## 防御策略

### 1. 输入验证与过滤

```python
class ContextDefenseLayer:
    def __init__(self):
        self.input_guard = InputGuard()
        self.content_filter = ContentFilter()
        self.sanitizer = ContextSanitizer()
    
    def defend(self, user_input: str, current_context: Context) -> DefendedInput:
        # 1. 检测提示词注入
        injection_signals = self.input_guard.detect_injection(user_input)
        if injection_signals:
            return DefendedInput(
                original=user_input,
                sanitized=self._remove_injection(user_input),
                warnings=injection_signals,
                blocked=False
            )
        
        # 2. 过滤有害内容
        filtered = self.content_filter.filter(user_input)
        
        # 3. 验证上下文一致性
        context_issues = self._check_context_consistency(
            filtered, 
            current_context
        )
        
        return DefendedInput(
            original=user_input,
            sanitized=filtered,
            warnings=context_issues,
            blocked=False
        )
```

### 2. 分层隔离机制

```python
class LayeredContextIsolation:
    """
    上下文分层隔离：
    Layer 0: 系统级约束（不可被覆盖）
    Layer 1: 工具定义（高优先级）
    Layer 2: 长期记忆（经过验证）
    Layer 3: 短期上下文（需要验证）
    Layer 4: 用户输入（需要过滤）
    """
    LAYER_PRIORITY = {
        'system_constraint': 100,
        'tool_definition': 90,
        'long_term_memory': 80,
        'verified_knowledge': 70,
        'recent_context': 60,
        'user_input': 50,
        'unverified_retrieval': 30
    }
    
    def merge(self, layers: Dict[str, Any]) -> MergedContext:
        # 按优先级排序
        sorted_layers = sorted(
            layers.items(),
            key=lambda x: self.LAYER_PRIORITY.get(x[0], 0),
            reverse=True
        )
        
        merged = {}
        for layer_name, content in sorted_layers:
            merged = self._merge_layer(merged, content, layer_name)
        
        return merged
```

### 3. 动态上下文清理

```python
class DynamicContextCleaner:
    def __init__(self, contamination_threshold: float = 0.3):
        self.threshold = contamination_threshold
    
    def should_clean(self, context: Context) -> bool:
        contamination_score = self._calculate_contamination(context)
        return contamination_score > self.threshold
    
    def clean(self, context: Context) -> Context:
        # 1. 移除低信任度内容
        cleaned_items = [
            item for item in context.items
            if self._get_trust_score(item) > 0.5
        ]
        
        # 2. 移除冲突内容（保留高信任度版本）
        cleaned_items = self._remove_conflicts(cleaned_items)
        
        # 3. 移除过时信息
        cleaned_items = self._remove_outdated(cleaned_items)
        
        return Context(items=cleaned_items)
```

## 失败模式与处理

| 失败模式 | 表现 | 解决方案 |
|---------|------|---------|
| 提示词注入 | 模型执行恶意指令 | 输入过滤 + 指令隔离 |
| 上下文膨胀 | 无关内容稀释关键信息 | 动态清理 + 优先级路由 |
| 信息冲突 | 模型决策不一致 | 冲突检测 + 消解策略 |
| 历史遗忘 | 重要上下文被意外清除 | 分层存储 + 选择性保留 |
